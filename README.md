### *GhostMan*

um jogo furtivo de labirinto reverso feito com [Gly Engine](https://github.com/gly-engine/gly-engine).

### *como rodar?*

precisa apenas de lua instalado e o script deve baixar a engine automaticamente

### *Desenvolvimento*

```bash
lua tool.lua dev
```

### *Build Web*

gera os arquivos estáticos em `dist/web`.

```bash
lua tool.lua build
```

## *Requisitos*

- Lua 5.1+
- (Opcional) `wget` ou `curl` para Windows
